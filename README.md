Your website is a Node.js + Express + Pug powered platform with a highly impressive UI that blends performance and aesthetics. It’s designed to showcase dance in a way that feels modern, interactive, and professional.

The Home page welcomes visitors with dynamic banners and smooth navigation. The About page shares the vision, mission, and story behind the dance community, highlighting instructors and values. The Gallery page showcases vibrant images and videos in a responsive layout, immersing users in the energy of dance. The Services page details offerings such as classes, workshops, online sessions, and event performances. Core base pages are designed with reusable Pug templates, ensuring consistency and efficiency across the site. The Contact page provides an interactive form with validation and location details, making communication effortless.

⚙️ Technical Highlights
- API Integration: All pages call APIs for dynamic content (e.g., fetching gallery items, services list, contact form submissions).
- Responsive UI: Optimized for desktop, tablet, and mobile.
- Reusable Components: Pug templates ensure DRY (Don’t Repeat Yourself) coding practices.
- Express Routing: Clean, modular routes for each page and API endpoint.
- Performance: Fast load times with efficient middleware and caching strategies.
